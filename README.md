# Client Gateway Microservice

## Dev

1. Clonar el repositorio
2. Instalar dependencias
3. Crear un archivo `.env` basado en el `.env.template`
4. Levantar base de datos `docker compose up -d`
5. Ejecutar `npm run start:dev`
